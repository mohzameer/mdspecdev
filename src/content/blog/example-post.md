# Welcome to the mdspec Blog!

This is our first post, powered entirely by a local markdown file living right inside the repository.

## Why local Markdown?

- **Simplicity**: No external database or headless CMS to manage.
- **Developer Experience**: Write content in your favorite code editor (like VS Code or Cursor).
- **Version Control**: Every post is versioned with Git along with the codebase.
